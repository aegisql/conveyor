About Conveyor Service

Conveyor Service provides REST and dashboard access to configured conveyors for placement, command execution, runtime inspection, and operational control.

Workspace Features

  • Conveyors: tree view of loaded conveyors and hierarchy with live state styling (running, suspended, stopped).
  • Watchers: watch IDs or full conveyor streams, view timeline updates, and manage watches from tags and output tabs.
  • Conveyor Details: runtime state, MBean information, metadata, and parameters.
  • Operations: writable parameter updates and MBean method invocation.
  • Parts / Static Parts / Commands: request forms with optional watch output.
  • Admin Actions: upload JARs, controlled reload, and tree delete for top-level conveyors.
  • Output: tabbed events view with per-tab JSONPath and cache limits.
